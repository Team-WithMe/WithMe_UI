# 📚 With Me design-system

with me 디자인 시스템 입니다.
<br />

## 📝 Installation

- [npm](https://www.npmjs.com/package/@with-me/design)
- [storybook](https://with-me-ui.netlify.app)

npm

```
npm i @with-me/design
```

yarn

```
yarn @with-me/design
```

<br />

### 📌 커스텀이 필요할 때

색상, 글꼴 등 커스텀이 필요한다면?

```
yarn @with-me/styles
```

- styles 패키지를 받아 sass의 변수를 오버라이딩 하면 됩니다
- 문서 작업 진행 중..
