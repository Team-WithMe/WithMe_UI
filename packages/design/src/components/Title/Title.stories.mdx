# 제목

## 색상

- 제목의 색상을 **color** 값으로 지정할 수 있습니다.
- default 값은 deep-gray (#222) 입니다.
- color = `primary` | `secondary` | `danger` | `success` | `deep-gray` | `greyish` | `light-gray` | `white`

## 사이즈

- 제목의 사이즈를 **size** 값으로 지정할 수 있습니다.
- default 값은 middle (2.1875rem = 35px) 입니다.
- size = `large` | `middle` | `small`
	- largem: **2.5rem = 40px**
	- middle: **2.1875rem = 35px**
	- small: **1.875rem = 30px**

<Canvas>
	<Story name="Size">
		<Title size="large">40 글씨 / large</Title>
		<Title size="middle">35 글씨 / middle</Title>
		<Title size="small">30 글씨 / small</Title>
	</Story>
</Canvas>

<br />

## 두께

- 제목의 두께를 **weight** 값으로 지정할 수 있습니다.
- default 값은 regular (400) 입니다.
- weight = `thin` | `light` | `regular` | `medium` | `bold`
	- thin: **100**
	- light: **300**
	- regular: **400**
	- medium: **500**
	- bold: **700**
