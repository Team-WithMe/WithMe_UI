## 색상

- 텍스트의 색상을 **color** 값으로 지정할 수 있습니다.
- default 값은 deep-gray (#222) 입니다.
- color = `primary` | `secondary` | `danger` | `success` | `deep-gray` | `greyish` | `light-gray` | `white`

## 사이즈

- 텍스트의 사이즈를 **size** 값으로 지정할 수 있습니다.
- default 값은 base (1rem = 16px) 입니다.
- size = `sm` | `base` | `lg` | `xl` | `xxl` | `xxxl`
	- sm: **0.875rem = 14px**
	- base: **1rem = 16px**
	- lg: **1.125rem = 18px**
	- xl: **1.25rem = 20px**
	- xxl: **1.375rem = 22px**
	- xxxl: **1.5rem = 24px**

	</Story>
</Canvas>

<br />

## 두께

- 텍스트의 두께를 **weight** 값으로 지정할 수 있습니다.
- default 값은 regular (400) 입니다.
- weight = `thin` | `light` | `regular` | `medium` | `bold`
	- thin: **100**
	- light: **300**
	- regular: **400**
	- medium: **500**
	- bold: **700**
