# Input

&lt;Input /&gt; 컴포넌트 사용법 정리

<br />

## 사이즈

- Input의 사이즈(높이) 를 **size** 값으로 지정할 수 있습니다.
- default 값은 middle (32px) 입니다.
- size = `large` | `middle` | `small`
	- large: **40px**
	- middle: **32px**
	- small: **24px**

<Canvas>
	<Story name="Size">
		<Input size="large" placeholder="large" />
		<Input size="middle" placeholder="middle" />
		<Input size="small" placeholder="small"  />
	</Story>
</Canvas>

<br />

## prefix와 suffix

- **prefix**, **suffix**를 이용해서 접두사와 접미사를 만들 수 있습니다.
- `type: ReactNode`

<Canvas>
	<Story name="PrefixAndSuffix">
		<Input prefix={<span>prefix</span>} />
		<Input suffix={<span>suffix</span>} />
		<Input prefix={<span>prefix</span>} suffix={<span>suffix</span>} />
		<Input prefix={<UserIcon />} />
		<Input suffix={<UserIcon />} />
	</Story>
</Canvas>

<br />

## password 모드

- **password**를 설정하면 password 모드가 적용됩니다.
- 오른쪽 아이콘을 이용해서 가려진 비밀번호를 볼 수 있습니다.
- `password`가 true 일 때,

	<Story name="Prefix / Suffix">

	</Story>
</Canvas>
